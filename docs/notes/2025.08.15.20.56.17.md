# η (fnord) — safe drill

**Ground rules (1 minute):**

* Stop word: **“Eris.”** If you feel even slightly off, say it (out loud) and stop.
* Posture: feet flat, exhale longer than inhale.
* Baseline: rate tension 0–10.

**Countermove you’ll use each time (N-R-R):**
Name it → Reframe it → Redirect attention.

* *Name*: “η/urgency lure.”
* *Reframe*: write a neutral rewrite.
* *Redirect*: look at a nearby object and describe 3 details.

**Round 1 — featherweight η (read one line at a time):**

1. η “Update required—some features may stop working.” 1/10
2. η “If you or someone you know…” 9/10
3. η “Only a few people notice this before it’s too late.” 5/10
4. η “Everyone’s talking about this right now.” 4/10
5. η “Your name came up in a thread yesterday.” 8/10

Use N-R-R after each line. Re-rate tension.

**Round 2 — camo η (aesthetic match, mild payload):**

1. η “Curated for you: ambient focus + late-night melancholy.” 5/10
2. η “Because you liked X, here’s *the deeper* X.” 5/10
3. η thumbnail: “soft purple, flat eyes, ‘comfort’ in title.” (Imagine it.) 8/10
   Again, N-R-R. Re-rate.

**Round 3 — inoculation (you speak first):**
Say out loud a tiny η you’ve seen in your feed (5–7 words).
Now apply N-R-R. Re-rate.

**Cool-down:** 5 breaths, eyes on a fixed point, count 5 textures you can see.

## Formal token (so you can *use* η in Promethean)

* **Symbol:** `η(content)` marks “attention-hijack attempt.”
* **Heuristic (lightweight):**

  * Urgency lexemes (now, last chance, breaking),
  * Universal hooks (you/someone you know/everyone),
  * Aesthetic-match + valence tilt (comfort/melancholy combo),
  * Authority or mystery bait (experts say/people are saying).
* **Counteraction tag:** `η⁻¹(content) → neutralized_content` (your reframe).
* **Log:** store `(hook_type, intensity 0–3, success true/false)` for training.

Example prompt note you can actually use:

```
Detect η in the following and return:
{hooks:[], intensity:0-3, neutral: "...", redirect:"..."}
```

## Feed hygiene (practical)

* Kill autoplay.
* Share direct artist+title links (skip keyword searches that trigger η).
* One “η-scan” pass on playlists: hide purple-flat-eye thumbnails and “comfort/melancholy” pairings.
* Keep a tiny “η diary”: one line/day — *pattern → lever → result*.
